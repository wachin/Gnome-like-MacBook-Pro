Installation using Python

Requirements: pip3

The best way to receive the most up-to-date version of Mesonbuild.

Install as a local user (recommended):

$ pip3 install --user meson

or:

Install as root:

$ pip3 install meson

Luego poner en la terminal:

echo "export PATH=~/.local/bin:$PATH" >> ~/.bashrc

reiniciar el ordenador para que el sistema pueda reconocer esa ruta para los ejecutables all√≠ presentes


Para compilar

$ cd /path/to/source/root
$ meson builddir && cd builddir
$ meson compile
$ meson test
